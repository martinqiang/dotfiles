# Notes for setup

Apps to install:
- Neovim
    - Use install-nvim.sh
    - Install plugins
        - `:PlugInstall`
        - `:F` to download FZF
        - `:Mason` to install LSP
- TMUX
    - Install plguins
        - Download [TMUX Plugin Manager](https://github.com/tmux-plugins/tpm)
        - Use `pref` + `I` to fetch plugins in tmux
